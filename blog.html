What is the Document Object Model (DOM)?

The DOM is a system that shows the structure of a webpage in a way that computers can understand. It lets programmers use code to change upadate style of the webpage without needing to reload it. This means they can add new content, respond to user actions, and make the page interactive


How do you select an element from the DOM?

We can use JavaScript methods, such as getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll() to select elements from the DOM


What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a programming technique in the context of the DOM that allows for more efficient event management by leveraging the event bubbling phase. Instead of adding an event listener to each individual child element, programmers can add a single event listener to a common parent element. This parent element listens for events that propagate up from its children. This approach is beneficial because it simplifies code, enhances maintainability, improves performance, and is particularly useful for handling dynamic content


How do you manipulate an element's attributes and styles using the DOM?

We can modify an element's attributes and styles in the DOM using JavaScript by selecting the element and utilizing various methods. We can manage attributes with getAttribute(), setAttribute(), removeAtrribute(), while the style property and classList so we can adjust CSS styles and classes. This allows us to dynamically alter the appearance and functionality of web elements.
